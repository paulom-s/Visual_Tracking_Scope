# Visual_Tracking_Scope
Visual Object Tracking software for Scope (Celestron NexStar 127SLT) by connecting it and an DSLR Camera (Canon EOS 100D) to the laptop.
Developping it on both Windows and Linux in order to build a small computer with a Raspberry PI or an old laptop with a Light Linux Distribution with or without GUI and put it on or inside of the scope.
